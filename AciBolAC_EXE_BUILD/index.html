Acero para losas y viguetas

Módulo especializado para calcular acero en losas unidireccionales, losas macizas por metro, viguetas/nervios y losas reticulares. Sirve para acero positivo o negativo.

Aportes adicionales de acero

La segunda capa se toma como una barra adicional por cada vigueta o nervio. Su aporte por metro se calcula con la separación entre viguetas/nervios.

Resultado de acero en losa

Ingresa el momento, tipo de losa y diámetros permitidos.
Cuando eliges momento por vigueta/nervio, el programa toma el Mu directamente para esa vigueta o nervio. Cuando eliges momento por metro, lo convierte usando la separación tributaria.

Acero por momento

Calcula el acero requerido As para una sección rectangular con refuerzo simple. Para vigas recomienda número de barras; para losas recomienda diámetro a una separación dada.

Este dato solo se usa cuando la recomendación es como losa/franja. En vigas no participa en el cálculo.

Armadura en 2 capas para vigas

Resultado de acero

Ingresa el momento, sección y diámetros permitidos.
La separación solo aplica para losa/franja. Para viga se usa el número máximo de barras y, si activas la opción, barras de ayuda en segunda capa.
